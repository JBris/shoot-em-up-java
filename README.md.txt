This program executes the game, Bionic Pilot. 

## Keyboard Controls ##
  
Up Arrow - Player Up; Menu Up
Down Arrow - Player Down; Menu Down
Left Arrow - Player Left; Menu Left
Right Arrow - Player Right; Menu Right
Attack - Space
Special Attack - X
Dash - Z
P - Pause
Enter - Confirm
Escape - Exit

## Mouse Controls ##

Left Click - Shoot
Right Click - Deflect
Mouse Movement - Player Movement
